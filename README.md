# Интернет магазин мотоциклов "Moto-X"
Описание:  
язык: PHP.  
фреймворк: Laravel 10.  


# Для развертывания

```
composer install
```
```
php artisan key:generat
```
```
php artisan storage:link
```

```
php artisan cache:clear
```
```
php artisan migrate
```

```
npm config set strict-ssl false
```
```
npm install
```

```
php artisan serve
```
```
npm run dev
```
